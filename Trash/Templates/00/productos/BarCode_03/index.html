Tipos de código más utilizados

Lineales (EAN / UPC / Code 128)

  • EAN-13 Retail general, 13 dígitos (muy común en Europa y LATAM).
  • EAN-8 Envases pequeños, 8 dígitos.
  • UPC-A Retail EE. UU./Canadá, 12 dígitos.
  • UPC-E UPC comprimido para productos de tamaño reducido.
  • Code 128 Alfanumérico, alta densidad, muy usado en logística y etiquetas internas.

2D (QR)

  • QR estándar URLs, IDs internos, payloads para apps móviles.
  • QR ficha producto Enlace a ficha técnica / detalle comercial.
  • QR interno Codifica SKU/variant_id para procesos de bodega y POS.
