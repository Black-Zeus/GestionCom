Lineales (EAN / UPC / Code 128)

  • EAN-13 Código más utilizado en retail europeo/latinoamericano (13 dígitos).
  • EAN-8 Variante corta (8 dígitos), para envases pequeños.
  • UPC-A Estándar predominante en EE. UU./Canadá (12 dígitos).
  • UPC-E Versión comprimida de UPC-A para empaques reducidos.
  • Code 128 Código alfanumérico de alta densidad, común en logística y etiquetas internas.

Códigos 2D (QR)

  • QR estándar Soporta URL, texto, identificadores internos, payloads para apps móviles, etc.
  • QR URL Enlace a ficha de producto, catálogo web o landing de soporte.
  • QR interno Puede codificar IDs internos de producto/variante para uso en almacén.
