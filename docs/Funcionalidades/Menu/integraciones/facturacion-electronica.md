# Facturación electrónica

**Ruta:** `/integrations/electronic-billing`  
**Permiso:** `DTE_ACCESS` / `DTE_VIEW` / `DTE_CONFIG_MANAGE`  
**Componente:** `AdminElectronicBilling`  
**Estado:** ✅ Implementado

## Misión

Configuración e integración con el proveedor de Documentos Tributarios Electrónicos (DTE). Gestiona las credenciales de conexión, los folios disponibles por tipo de documento y el estado de la integración con el Servicio de Impuestos Internos (SII).

## Funcionalidades implementadas

- Configuración de credenciales del proveedor DTE
- Estado de la integración (conectado / desconectado)
- Gestión de folios disponibles por tipo de documento electrónico
- Consulta del historial de documentos enviados al SII
- Indicador de documentos pendientes de envío o con error
