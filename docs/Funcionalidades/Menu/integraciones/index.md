**Permiso de visibilidad:** `DTE_ACCESS` / `DTE_VIEW` / `DTE_CONFIG_MANAGE`

Módulo de integración con servicios externos. Actualmente contiene la configuración de la facturación electrónica (DTE) para la emisión de documentos tributarios electrónicos.
