# Módulo: Configuración

**Ruta raíz:** `/config/*`  
**Permiso de visibilidad:** `SETTINGS_VISIBLE`  
**Estado:** ⚠️ Parcial

## Descripción

Módulo de configuración del sistema. Centraliza todos los parámetros operativos de la plataforma: datos de la empresa, impuestos, métodos de pago, unidades de medida, notificaciones y logs del sistema.
