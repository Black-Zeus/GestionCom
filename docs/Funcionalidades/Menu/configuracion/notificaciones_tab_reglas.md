# Tab: Reglas de notificación

**Pertenece a:** [Configuración de notificaciones](notificaciones.md)  
**Ruta:** `/config/notifications` (tab activo: `rules`)  
**Estado:** ✅ Implementado

## Misión

Reglas de emisión automática de notificaciones. Define bajo qué condiciones del sistema se dispara cada tipo de notificación, con control de frecuencia y prioridad.

## Funcionalidades implementadas

- Listado de reglas por origen del evento
- Creación y edición: origen, severidad forzada, prioridad mínima
- Límite máximo de notificaciones por usuario por día
- Flags: emitir en bandeja interna, emitir como push
- Activar / desactivar regla
