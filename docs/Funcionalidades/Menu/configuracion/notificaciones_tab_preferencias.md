# Tab: Preferencias de notificación

**Pertenece a:** [Configuración de notificaciones](notificaciones.md)  
**Ruta:** `/config/notifications` (tab activo: `preferences`)  
**Estado:** ✅ Implementado

## Misión

Preferencias individuales de recepción de notificaciones por usuario. Permite al administrador configurar qué canales de entrega prefiere cada usuario y si tiene silenciadas las notificaciones temporalmente.

## Funcionalidades implementadas

- Listado de preferencias por usuario
- Edición: flag bandeja interna, flag push
- Configuración de silencio temporal (`muted_until`)
- Activar / desactivar preferencias del usuario
