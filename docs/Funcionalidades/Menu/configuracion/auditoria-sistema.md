# Auditoría del sistema (Configuración)

**Ruta:** `/config/system-audit`  
**Permiso:** `SETTINGS_VISIBLE`  
**Componente:** — (sin componente asignado)  
**Estado:** 🚧 En desarrollo

## Misión

Vista de auditoría técnica del sistema desde el módulo de Configuración. Muestra el historial de cambios en la configuración del sistema (parámetros, impuestos, permisos) con trazabilidad por usuario.

## Funcionalidades planificadas

- Log de cambios en configuración del sistema
- Filtros por tipo de configuración y usuario
- Comparativa de valores anteriores vs. actuales
