# Parámetros del sistema

**Ruta:** `/config/system-parameters`  
**Permiso:** `SYSTEM_PARAMETERS_ACCESS`  
**Componente:** `AdminSystemParameterMaintainers` → `SystemParameterMaintainers`  
**Estado:** ✅ Implementado

## Misión

Gestión de los estados base que usan los distintos módulos del sistema (estados de documentos, de transferencias, de solicitudes, etc.). Permite personalizar las etiquetas visibles y los colores/iconos asociados a cada estado.

## Funcionalidades implementadas

- Listado de estados agrupados por grupo funcional
- Creación y edición: grupo, código interno, nombre visible, color, icono, orden
