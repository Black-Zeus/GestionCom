# Unidades de medida

**Ruta:** `/config/measurement-units`  
**Permiso:** `MEASUREMENT_UNITS_ACCESS` o `MEASUREMENT_UNITS_MANAGE`  
**Componente:** `AdminMeasurementUnits`  
**Estado:** ✅ Implementado

## Misión

Catálogo global de unidades de medida del sistema (kg, g, L, mL, unidad, caja, docena, etc.). Las unidades aquí definidas se usan para configurar unidades de inventario, compra y venta en los productos.

## Funcionalidades implementadas

- Listado de unidades con símbolo, tipo y estado
- Creación y edición: nombre, símbolo, tipo de unidad
- Activar / desactivar unidad de medida
