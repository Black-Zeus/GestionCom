# Configuración de notificaciones

**Ruta:** `/config/notifications`  
**Permiso:** `NOTIFICATION_SETTINGS_ACCESS` o `NOTIFICATION_SETTINGS_MANAGE`  
**Componente:** `AdminNotificationSettings` → `NotificationSettingsMaintainers`  
**Estado:** ✅ Implementado

## Misión

Gestión del sistema de notificaciones interno. Configura los tipos de notificación existentes, las reglas de emisión (cuándo y con qué prioridad se emiten) y las preferencias de recepción por usuario.

| Tab | Descripción | Detalle |
