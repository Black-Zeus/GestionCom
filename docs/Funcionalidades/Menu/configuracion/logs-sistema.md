# Logs del sistema

**Ruta:** `/config/system-logs`  
**Permiso:** `SETTINGS_VISIBLE`  
**Componente:** — (sin componente asignado)  
**Estado:** 🚧 En desarrollo

## Misión

Visualización de los logs de aplicación del backend directamente desde la interfaz web. Permite al administrador técnico consultar errores, warnings y trazas de ejecución sin acceder al servidor.

## Funcionalidades planificadas

- Listado de entradas de log con nivel (INFO/WARNING/ERROR)
- Filtros por nivel, fecha y módulo de origen
- Visualización de stack trace expandible
- Descarga del archivo de log del período
