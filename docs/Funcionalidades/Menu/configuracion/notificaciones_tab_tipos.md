# Tab: Tipos de notificación

**Pertenece a:** [Configuración de notificaciones](notificaciones.md)  
**Ruta:** `/config/notifications` (tab activo: `types`)  
**Estado:** ✅ Implementado

## Misión

Catálogo de los tipos de notificación que puede emitir el sistema (alerta de stock crítico, solicitud de aprobación, reposición de caja chica, etc.). Define la severidad por defecto y la acción sugerida al usuario.

## Funcionalidades implementadas

- Listado de tipos de notificación con severidad y estado
- Creación y edición: nombre, severidad (info/warning/error/critical), acción sugerida
- Flag: visible para el usuario en la bandeja
- Activar / desactivar tipo
