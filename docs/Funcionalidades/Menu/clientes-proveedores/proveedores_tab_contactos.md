# Tab: Contactos del proveedor

**Pertenece a:** [Listado de proveedores](proveedores.md)  
**Ruta:** `/suppliers` (tab activo: `contacts`) o `/suppliers/contacts?supplier_id=X`  
**Componente:** `AdminSupplierContacts`  
**Estado:** ✅ Implementado

## Misión

Registro de las personas de contacto de cada proveedor con sus datos de comunicación y rol dentro de la organización.

## Funcionalidades implementadas

- Listado de contactos filtrado por proveedor
- Creación y edición: nombre, cargo, email, teléfono, móvil
- Flag: contacto principal
- Activar / desactivar contacto
