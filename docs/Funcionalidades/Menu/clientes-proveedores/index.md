# Módulo: Clientes y proveedores

**Ruta raíz:** `/customers`, `/suppliers`  
**Permiso de visibilidad:** `CUSTOMERS_VISIBLE`  
**Estado:** ✅ Implementado

## Descripción

Módulo de gestión de la cartera comercial. Centraliza el mantenimiento de clientes (con sus personas autorizadas y condiciones de crédito) y de proveedores (con contactos, direcciones y productos asociados).
