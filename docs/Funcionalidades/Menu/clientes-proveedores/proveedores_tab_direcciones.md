# Tab: Direcciones del proveedor

**Pertenece a:** [Listado de proveedores](proveedores.md)  
**Ruta:** `/suppliers` (tab activo: `addresses`) o `/suppliers/addresses?supplier_id=X`  
**Componente:** `AdminSupplierAddresses`  
**Estado:** ✅ Implementado

## Misión

Registro de las direcciones comerciales y de despacho asociadas a un proveedor, permitiendo gestionar múltiples ubicaciones.

## Funcionalidades implementadas

- Listado de direcciones del proveedor
- Creación y edición: dirección, ciudad, región, país
- Flag: dirección principal
- Activar / desactivar dirección
