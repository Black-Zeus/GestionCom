# Tab: Cuentas bancarias

**Pertenece a:** [Bancos y cuentas bancarias](bancos-cuentas.md)  
**Ruta:** `/finance/banking` (tab activo: `accounts`)  
**Estado:** ✅ Implementado

## Misión

Registro de las cuentas bancarias de la empresa. Cada cuenta tiene un saldo inicial y un saldo actual que se actualiza con los movimientos registrados.

## Funcionalidades implementadas

- Listado de cuentas con banco, número, saldo actual y estado
- Creación y edición: banco, número de cuenta, saldo inicial, saldo actual
- Activar / desactivar cuenta
