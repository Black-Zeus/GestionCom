# Módulo: Finanzas

**Ruta raíz:** `/finance/*`  
**Permiso de visibilidad:** `FINANCE_VISIBLE`  
**Estado:** ✅ Implementado

## Descripción

Módulo de administración financiera. Gestiona el catálogo de bancos y cuentas bancarias (incluyendo parámetros de conciliación), el catálogo de monedas con tasas de cambio, y los convenios comerciales con clientes o instituciones.
