# Convenios

**Ruta:** `/finance/agreements`  
**Permiso:** `AGREEMENTS_ACCESS` o `AGREEMENTS_MANAGE`  
**Componente:** `AdminAgreements`  
**Estado:** ✅ Implementado

## Misión

Gestión de convenios comerciales con clientes, instituciones o entidades (ej: convenios de empresa, convenios con gremios, acuerdos de descuento institucional). Define las condiciones especiales de precio o crédito que se aplican a un grupo de clientes bajo un convenio activo.

## Funcionalidades implementadas

- Listado de convenios con nombre, vigencia y estado
- Creación y edición de convenio: nombre, descripción, fechas de vigencia
- Asociación de clientes o grupos al convenio
- Definición de condiciones (descuento %, lista de precios especial, plazo de crédito)
- Activar / desactivar convenio
