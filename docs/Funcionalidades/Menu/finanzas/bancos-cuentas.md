# Bancos y cuentas bancarias

**Ruta:** `/finance/banking`  
**Permiso:** `FINANCE_MAINTAINERS_ACCESS` o `FINANCE_MAINTAINERS_MANAGE`  
**Componente:** `AdminFinanceMaintainers` → `FinanceMaintainers (tab: banks)`  
**Estado:** ✅ Implementado

## Misión

Mantenedor del catálogo de bancos y de las cuentas bancarias de la empresa. Incluye la configuración de parámetros de conciliación bancaria automática por cuenta.

## Funcionalidades implementadas

- Listado de bancos con país y estado
- Creación y edición de banco: nombre, país
- Activar / desactivar banco
