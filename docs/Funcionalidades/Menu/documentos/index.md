# Módulo: Documentos

**Ruta raíz:** `/documents/*`  
**Permiso de visibilidad:** `DOCUMENTS_VISIBLE`  
**Estado:** ⚠️ Parcial

## Descripción

Módulo de gestión documental comercial. Centraliza la configuración de tipos y series de documentos tributarios, las plantillas de salida, la consulta de documentos emitidos y el flujo de cambios, devoluciones y notas de crédito.
