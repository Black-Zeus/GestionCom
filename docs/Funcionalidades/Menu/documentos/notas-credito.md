# Notas de crédito

**Ruta:** `/documents/returns/credit-notes`  
**Permiso:** `DOCUMENTS_VISIBLE`  
**Componente:** — (sin componente asignado en router)  
**Estado:** 🚧 En desarrollo

## Misión

Consulta y gestión del archivo de notas de crédito emitidas. Las notas de crédito se generan automáticamente al procesar devoluciones de documentos tributarios (facturas, boletas electrónicas) o pueden emitirse manualmente para corregir errores de facturación.

## Funcionalidades planificadas

- Listado de notas de crédito emitidas con filtros por fecha, cliente y documento referenciado
- Visualización del detalle y documento origen
- Re-impresión / re-envío de nota de crédito
- Gestión de notas de crédito pendientes de aplicar
