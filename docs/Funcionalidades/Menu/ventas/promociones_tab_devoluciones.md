# Tab: Devoluciones (motivos)

**Pertenece a:** [Mantenedor de Promociones](promociones.md)  
**Ruta:** `/sales/promotions` (tab activo: `returns`)  
**Estado:** ✅ Implementado

## Misión

Catálogo de motivos de devolución utilizados por el flujo de cambios y devoluciones. Define las reglas operativas asociadas a cada motivo (plazo máximo, si afecta stock, si permite cambio o reembolso).

## Funcionalidades implementadas

- Listado de motivos de devolución con estado activo/inactivo
- Creación y edición de motivo con:
  - Días máximos permitidos desde la venta
  - Flag: requiere aprobación
  - Flag: afecta stock
  - Flag: permite cambio / reembolso
- Activación y desactivación de motivos

> Este mismo contenido es accesible también desde `/documents/returns/reasons` (módulo Documentos).
