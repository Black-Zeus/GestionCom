# Dashboard principal

**Ruta:** `/dashboard`  
**Permiso:** `HOME_VISIBLE`  
**Componente:** `Home`  
**Estado:** ✅ Implementado

## Misión

Página de aterrizaje del sistema tras el login. Presenta un resumen visual del estado operativo del negocio: indicadores de caja del día, ventas recientes, alertas de stock y accesos rápidos a las operaciones frecuentes.

## Funcionalidades implementadas

- Tarjetas resumen con KPIs del día (ventas, cobros, caja)
- Accesos rápidos a módulos frecuentes
- Panel de alertas activas (stock crítico, documentos pendientes)
