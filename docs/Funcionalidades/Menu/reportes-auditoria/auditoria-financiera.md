## Misión

Reporte de auditoría de operaciones financieras: conciliaciones bancarias, movimientos de cuentas, diferencias detectadas y acciones correctivas aplicadas. Orientado a contabilidad y control interno.

## Funcionalidades planificadas

- Resumen de conciliaciones del período

- Diferencias entre sistema y estado bancario
- Exportación para auditor externo
