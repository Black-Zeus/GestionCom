# Módulo: Reportes de auditoría

**Ruta raíz:** `/reports/audit/*`, `/reports/financial/*`  
**Permiso de visibilidad:** `AUDIT_VISIBLE`  
**Estado:** 🚧 En desarrollo

## Descripción

Módulo de reportería de auditoría y control interno. Provee acceso a informes de auditoría financiera, trazabilidad de acciones del sistema y actividad de usuarios. Ninguno de los tres reportes tiene componente implementado aún.
