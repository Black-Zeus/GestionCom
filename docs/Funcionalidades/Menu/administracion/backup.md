# Backup y restauración

**Ruta:** `/admin/backup`  
**Permiso:** `ADMIN_VISIBLE`  
**Componente:** — (sin componente asignado)  
**Estado:** 🚧 En desarrollo

## Misión

Gestión de copias de seguridad de la base de datos y archivos del sistema. Permite al administrador generar backups manuales, programar backups automáticos y restaurar desde un punto de recuperación previo.

## Funcionalidades planificadas

- Listado de backups disponibles con fecha, tamaño y tipo
- Generación de backup manual bajo demanda
- Configuración de backup automático programado
- Descarga del archivo de backup
- Restauración desde backup con confirmación de acción destructiva
