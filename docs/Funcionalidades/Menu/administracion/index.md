# Módulo: Administración

**Ruta raíz:** `/admin/*`  
**Permiso de visibilidad:** `ADMIN_VISIBLE`  
**Estado:** ⚠️ Parcial

## Descripción

Módulo de administración del sistema. Gestiona los usuarios, roles y permisos, la configuración operativa de caja POS, las denominaciones de efectivo, las categorías de caja chica y los respaldos.
