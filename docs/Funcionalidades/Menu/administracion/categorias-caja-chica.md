# Categorías de caja chica

**Ruta:** `/admin/cash/petty-cash-categories`  
**Permiso:** `PETTY_CASH_CATEGORIES_ACCESS` / `PETTY_CASH_ADMIN_ACCESS` / `PETTY_CASH_CATEGORIES_MANAGE`  
**Componente:** `AdminPettyCashCategories`  
**Estado:** ✅ Implementado

## Misión

Catálogo de categorías de gasto para caja chica. Las categorías se usan al registrar un gasto de caja chica para clasificarlo contablemente (ej: Útiles de oficina, Movilización, Alimentación).

## Funcionalidades implementadas

- Listado de categorías con nombre, descripción y estado
- Creación y edición: nombre, descripción, cuenta contable opcional
- Activar / desactivar categoría
