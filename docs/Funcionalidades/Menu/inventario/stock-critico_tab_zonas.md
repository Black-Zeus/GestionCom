# Tab: Zonas de bodega

**Pertenece a:** [Stock crítico y reposición](stock-critico.md)  
**Ruta:** `/inventory/stock/critical` (tab activo: `zones`)  
**Estado:** ✅ Implementado

## Misión

Gestión de las zonas de almacenamiento dentro de cada bodega. Una zona es una división lógica o física de la bodega (ej: Zona Fría, Zona Seca, Mezzanine).

## Funcionalidades implementadas

- Listado de zonas por bodega
- Creación y edición: nombre, descripción, tipo, bodega padre
- Contador de ubicaciones dentro de la zona
- Activar / desactivar zona
