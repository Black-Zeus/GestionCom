# Tab: Ubicaciones de bodega

**Pertenece a:** [Stock crítico y reposición](stock-critico.md)  
**Ruta:** `/inventory/stock/critical` (tab activo: `zone-locations`)  
**Estado:** ✅ Implementado

## Misión

Gestión de las ubicaciones físicas (slots) dentro de una zona de bodega. Permite mapear el almacén a nivel de pasillo, estante y posición para un control preciso de la ubicación del stock.

## Funcionalidades implementadas

- Listado de ubicaciones por zona
- Creación y edición: nombre, descripción, tipo de ubicación, orden
- Activar / desactivar ubicación
