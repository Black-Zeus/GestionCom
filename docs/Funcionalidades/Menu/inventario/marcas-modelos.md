# Marcas y modelos de productos

**Ruta:** `/inventory/products/brands-models`  
**Permiso:** `PRODUCT_BRAND_MODELS_ACCESS` o `PRODUCT_BRAND_MODELS_MANAGE`  
**Componente:** `AdminProductSupportMaintainers` → `ProductBarcodeMaintainers`  
**Estado:** ✅ Implementado

## Misión

Catálogo de marcas comerciales y sus modelos asociados. Se usa para clasificar productos por fabricante/marca y facilitar la búsqueda y filtrado en el catálogo.

## Funcionalidades implementadas

- Listado de marcas con sus modelos
- Creación y edición de marca: nombre, descripción
- Creación y edición de modelo vinculado a una marca
- Activar / desactivar marca o modelo
