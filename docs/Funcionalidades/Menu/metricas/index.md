# Módulo: Métricas e indicadores

**Ruta raíz:** `/metrics/*`  
**Permiso de visibilidad:** `METRICS_VISIBLE`  
**Estado:** ✅ Implementado

## Descripción

Módulo de visualización de métricas e indicadores de negocio. Presenta dashboards analíticos con gráficos interactivos (ApexCharts) para los principales dominios del sistema: ventas, inventario, caja y clientes.
