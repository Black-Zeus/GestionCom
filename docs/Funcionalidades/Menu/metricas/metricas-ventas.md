# Métricas de ventas

**Ruta:** `/metrics/sales`  
**Permiso:** `METRICS_VISIBLE`  
**Componente:** `SalesMetrics`  
**Estado:** ✅ Implementado

## Misión

Dashboard analítico de ventas. Presenta indicadores clave de rendimiento comercial con gráficos interactivos, comparativas de períodos y desglose por sucursal, vendedor o producto.

## Funcionalidades implementadas

- KPIs de ventas: total del período, ticket promedio, número de transacciones
- Gráfico de ventas diarias / semanales (serie temporal)
- Comparativa vs. período anterior
- Desglose por sucursal
- Exportación CSV del dataset subyacente
- Selector de rango de fechas
